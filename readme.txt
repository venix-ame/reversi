Reversi - 黑白棋游戏
author: venix - March 2023

简介:
这是我做棋类对弈游戏的实验.
应用程序分为四个模块, base模块负责实现黑白棋对弈的抽象数据结构, robot模块负责人机对弈的实现,
shell模块负责实现一个以终端为基础的交互界面, gui模块负责实现一个以GTK+为基础的图形交互界面.
应用程序使用C++ 17和ANSI-C 17编写, 在ArchLinux下编译和测试.

编译:
make        -   编译主程序.
make clear  -   清除二进制文件.
make clear2 -   清除二进制文件和vscode调试用临时文件.
